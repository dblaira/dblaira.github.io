# Caption Library

Use short composition-first captions. They should name what the visual is doing, not how it is implemented.

## World / Atmosphere

- World before interface
- Background carries the rule
- Atmosphere sets the hierarchy
- Pattern as environment
- The field speaks first

## Foreground / Subject

- Object with tension
- Figure as interpreter
- Presence before language
- The subject anchors the system
- Gesture creates meaning

## Product / Mass / Scale

- Mass carries meaning
- Scale creates authority
- Weight before words
- The object is the noun
- The eye obeys size

## Chrome / Navigation

- Navigation as whisper
- Chrome inside the atmosphere
- Utility without intrusion
- Interface in a low voice
- Control after desire

## Structure / Relation

- Cluster before label
- Relation through proximity
- Silence around the focal point
- One center of gravity
- Hierarchy without explanation

## Reverse-Spec Prompts

Pair captions with prompts like:

- What is doing the heavy lifting here?
- What does the eye understand before reading?
- What is background and what is foreground?
- What part is loud and what part stays silent?
- Which components are implied by this composition?
